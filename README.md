# BaltiLLM 

BaltiLLM is an open-source mini Language Model project for the Balti language.
The goal is to learn how LLMs work internally while contributing to the
preservation and digitization of the Balti language.

##  Goals
- Build a tokenizer for Balti
- Train a small transformer-based language model
- Learn end-to-end LLM development
- Enable community collaboration

##  Project Scope
This is an educational mini-LLM project.
It is NOT intended to compete with large models like GPT.
